# Delivery Artifacts

This directory stores iteration-scoped artifacts only.

## Subdirectories

- `iterations/`: iteration logs and baselines
- `quality-gates/`: gate reports and check outputs
- `phase-1-foundation/`: phase 1 task/review artifacts
- `phase-2-components/`: phase 2 task/review/design artifacts
- `knowledge/`: layered iteration memory (L1-L4 + snapshots)

## Do not place here

- evergreen component usage guides
- stable API reference docs
- long-term architecture overviews

Those belong to the root product-doc directories under `docs/admin-ui-framework/`.
